Production Diary – Week 15

10 February 2026

This week I worked on developing my GitHub Pages portfolio website. I focused on building the main sections, including About, Projects, Skills, and Contact. I also created a separate CS Maths & Tools page to demonstrate practical computer science maths concepts.

What I actually changed this week was improving the structure of my homepage with clear navigation and a more professional layout. I also added a Production Diary section to reflect on my learning process.

The achievement I am most proud of is successfully publishing my portfolio online. Seeing my work live through GitHub Pages helped me feel more confident about web development.

The most challenging part was understanding GitHub commits and how updates are pushed correctly. I also found it difficult at first to organise HTML content into clean sections.

From this work, I learned more about website structure, layout, and how basic JavaScript can be used for interactive tools. I also connected this to CS maths, especially binary representation, which is essential in computing.

Next week, I want to improve my portfolio by adding more projects and expanding my Maths & Tools page with at least one more interactive JavaScript feature.
